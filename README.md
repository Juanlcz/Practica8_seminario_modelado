# Practica8_seminario_modelado

Este proyecto presenta la simulación de un robot planar de 2 grados de libertad (GDL) desarrollado en Simscape Multibody. El objetivo principal es la integración de un sistema mecánico con actuadores eléctricos controlados en lazo cerrado.
